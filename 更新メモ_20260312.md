# 更新メモ（2026-03-12）

## 対象
- `index.html`

## 修正内容
- スマホで詳細モーダルを閉じた後にレイアウトが崩れる問題を修正
  - `body` のスクロールロック方式を `overflow` 切替から `position: fixed` + スクロール位置保存方式へ変更
  - モーダル終了時に元のスクロール位置へ復帰
  - モーダルオーバーレイに `overscroll-behavior: contain` を追加
- スマホで詳細モーダルの文字が小さい問題を修正
  - `@media(max-width:640px)` でモーダル本文・チェックリスト・注記のフォントサイズを引き上げ
  - モーダル余白、ヘッダー、閉じるボタンのサイズを見直し
  - 見出し・KPI・比較値の可読性を改善

- 詳細モーダルから戻った際に、ページ位置やレイアウトが不安定になりにくい
- スマホでの詳細テキスト閲覧性が改善し、操作ミスが減る
